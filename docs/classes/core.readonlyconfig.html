Class ReadOnlyConfig

ReadOnlyConfig class

ReadOnlyConfig is a utility class for use during runtime of the seagull application. This assumption enables caching of the config which should not change during runtime. Thus subsequent access does not include file system access. For testing purposes you can write into the returned reference and change the config. Usage of this behavior in production is discouraged.
